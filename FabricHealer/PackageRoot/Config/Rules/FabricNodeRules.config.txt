## This houses both Fabric Node restart and remove rules.

## First check if we are inside the run interval. If so, cut (!).
Mitigate() :- CheckInsideRunInterval(RunInterval=00:15:00), !.

Mitigate() :- GetRepairHistory(?repairCount, ?lastRunTime, RestartFabricNode), 
	?repairCount < 5,
	RestartFabricNode(MaxRepairs=5).
